gwst

Git worktrees, organized as workspaces.

A git workspace tool and worktree manager for AI agents and parallel development. Create task workspaces from issues, reviews, or presets without stepping on each other.

View on GitHub Quickstart

Why teams adopt gwst

Issue-driven work

Pick multiple GitHub issues in one pass and spin up dedicated workspaces per task.

Review bursts

When reviews pile up, create review workspaces in bulk and run AI agents safely.

Multi-repo changes

Templates bundle repositories into a single task workspace (pseudo-monorepo).

Safe cleanup

Guardrails prevent accidental deletion when worktrees are dirty or unpushed.

Install

Homebrew

brew tap tasuku43/gwst

brew install gwst

mise

mise use -g github:tasuku43/gwst

Direct download

Download the prebuilt binary and put gwst on your PATH.

Quickstart

1) gwst init
2) gwst repo get <repo>
3) gwst create --repo <repo>
4) gwst open / gwst rm

FAQ

Do I need gh?

Only for --issue and --review modes (GitHub).

What does gwst delete?

It removes worktrees and directories, and warns on dirty or unpushed work.
