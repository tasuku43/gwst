# Support

Thank you for using gion.  
Please choose the appropriate channel depending on the purpose of your inquiry.

---

## Questions (Usage / General help)

- GitHub Discussions:  
  https://github.com/tasuku43/gion/discussions

Before asking a question, please make sure to check:

- The README
- The `docs/` directory at the root of the project

Especially if you are using a **Coding Agent (e.g. Claude Code, Codex CLI, etc.)**,  
specifying the `docs/` directory as the investigation target and letting the agent explore
the specifications and behavior from there may help it arrive at an answer or identify the cause of an issue.

### Information to include when asking a question
- The command you executed
- Expected result / Actual result
- `gion --version`
- OS / Shell (e.g. macOS 14 / zsh)
- Relevant logs (please mask sensitive information)

## Feature Requests

- GitHub Issues (Feature request)
- Or GitHub Discussions:  
  https://github.com/tasuku43/gion/discussions

### Information to include in a feature request
- The problem you want to solve (who is affected and how)
- Expected UX (CLI examples, option ideas, etc.)
- Alternatives or workarounds (if any)
- Scope (MVP vs. future extensions)
- Compatibility impact and whether the change would be breaking

---

## Support Scope

- Responses are provided on a best-effort basis
- In-depth debugging or consulting for highly environment-specific issues is not guaranteed  
  (please provide clear reproduction steps or a minimal setup)
